# Problema 3 – Auditoría de Inventario y Reabastecimiento

Proyecto desarrollado para la **Fase 5 – Evaluación Final POA** del curso **Fundamentos de Programación (213022)** de la **Universidad Nacional Abierta y a Distancia (UNAD)**.

## Descripción

Este programa, desarrollado en **Python**, permite auditar un inventario almacenado en una matriz. Para cada artículo compara el **stock actual** con el **stock mínimo requerido** y calcula la cantidad exacta que debe solicitarse para reabastecer el inventario.

## Funcionalidades

- Registro de artículos en una matriz.
- Cálculo automático de la cantidad a solicitar.
- Uso de funciones para separar la lógica del programa.
- Recorrido del inventario mediante ciclos `for`.
- Generación de un informe con la cantidad de unidades a pedir por artículo.
- Resumen con el total de artículos evaluados y las unidades a solicitar.

## Tecnologías utilizadas

- Python 3

## Ejecución

1. Clonar el repositorio:

```bash
git clone https://github.com/patgeo117/problema3-fase5-unad.git
```

2. Ingresar a la carpeta del proyecto:

```bash
cd problema3-fase5-unad
```

3. Ejecutar el programa:

```bash
python auditoria_inventario.py
```
